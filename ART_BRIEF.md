# SD-Trainer 美术素材需求文档

## 项目背景

SD-Trainer（lora-scripts-next）是一个面向 AI 绘画爱好者和创作者的 **LoRA 模型训练工具**，通过浏览器 WebUI 操作。用户群体以国内二次元 AI 绘画社区为主，工具风格追求 **科技感 + 二次元亲和力**。

项目地址：https://github.com/wochenlong/lora-scripts-next

---

## 看板娘角色设定

| 项目 | 描述 |
|------|------|
| **年龄感** | 16–18 岁，少女感 |
| **人设方向** | 「青涩的 AI 训练指导老师」/「小向导」 — 年纪不大但认真教你的感觉 |
| **气质** | 亲和力强、温柔可靠、略带认真感，不是冷酷高冷型 |
| **主色调** | **洋红色 (Magenta)** 系，与 UI 品牌色统一 |
| **服装方向** | 科技 / 学院风混搭（实验服、制服、耳机、平板电脑等道具），体现「训练 / 教学」主题 |
| **关键词** | 原创、亲和、科技少女、小老师 |

> **注意**：角色需要为 **全新原创设计**，不可使用现有 IP 角色。

## 素材清单

### 素材 1：WebUI 看板娘 Logo ⭐ 必须

| 项目 | 说明 |
|------|------|
| **放置位置** | WebUI 首页正中央，项目名称「SD-Trainer」上方 |
| **画面内容** | 看板娘半身像或头像，表情自然友好，**不需要文字** |
| **尺寸** | **1024 × 1024 px**（正方形） |
| **格式** | PNG，**透明背景** |
| **显示说明** | 实际显示为 200 × 200 px，缩小后需仍然清晰好看 |

### 素材 2：Favicon 浏览器小图标 ⭐ 必须

| 项目 | 说明 |
|------|------|
| **放置位置** | 浏览器标签页左侧小图标 |
| **画面内容** | 看板娘的极简头像，或项目字母标识（如「ST」） |
| **尺寸** | **256 × 256 px** |
| **格式** | PNG，透明背景 |
| **显示说明** | 最小会缩到 16 × 16 px，越简洁越好 |

### 素材 3：GitHub 仓库首图横幅 ⭐ 必须

| 项目 | 说明 |
|------|------|
| **放置位置** | GitHub 仓库 README 最顶部，进入仓库第一眼看到的大图 |
| **尺寸** | **1280 × 720 px**（横版） |
| **格式** | PNG，**不透明背景** |

**画面布局建议**：

```
┌──────────────────────────────────────────┐
│                                          │
│   SD-Trainer            ┌──────────┐     │
│   ━━━━━━━━━━            │          │     │
│                         │  看板娘   │     │
│   AI LoRA Training      │  立绘    │     │
│   Made Easy             │  (半身)   │     │
│                         │          │     │
│   支持 Anima 训练        └──────────┘     │
│                                          │
│          ░░ 浅色渐变背景（白→淡洋红）░░     │
└──────────────────────────────────────────┘
```

- **左侧**：项目名称大号加粗 + 一行 slogan
- **右侧**：看板娘立绘（半身或全身）
- **背景**：浅色渐变（白 → 淡洋红），干净通透
